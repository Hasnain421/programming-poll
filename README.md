# Programming Poll

A simple Python program to conduct a poll asking people why they like programming.  
The responses are saved to a text file for later review.

---

## Features

- Ask users why they like programming.
- Save each response to a file (`programming poll.txt`).
- Stop the poll anytime by typing `quit`.
- Simple and beginner-friendly Python code.
